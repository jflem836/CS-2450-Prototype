# CS 2450 Prototype
This project is a high-fidelity prototype of a course planning and scheduling interface. It is built using HTML, CSS, and JavaScript, with no backend required.

## Current UI Features
- Course browsing and selection
- Add/remove course interactions
- Backup course saving
- Schedule visualization
- Interactive UI elements (dropdowns, buttons, tooltips)
- Filter popup UI
- Visual feedback (selected, saved, removal states)

## Run Options:

You can run locally by downloading, or click our github pages link to quickly view it online without any downloads required
### Github Pages Link:

https://jflem836.github.io/CS-2450-Prototype/

### Local Run Option 1
1. Download the all the files into a folder and open the folder.
2. Right-click on file named "index.html".
3. Click "Open With" → click "Web Browser".

### Local Run Option 2
1. Open the project folder in Visual Studio Code.
2. Install the Live Server extension if not previously installed.
3. Right-click "index.html" → click "Open with Live Server".

### Local Run Option 3
1. Download python if not previously downloaded.
2. Open main folder with all files and in file explorer, type cmd in the top search bar while in main folder and press enter.
3. Run the command `python -m http.server` in the command terminal.
4. Open your browser and go to: http://localhost:8000
